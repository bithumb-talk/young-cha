## 👋 프로젝트 소개 👋 

YoungCha는 가상자산과 관련된 소식 및 이벤트를 쉽게 접할 수 있으며, 실시간 시세 정보를 확인하여 빠른 정보 공유 및 의미 있는 투자정보를 공유할 수 있는 서비스 입니다.

- 팀명 **Youngcha** : 투자한 자산이 상승하기를 응원하는 의미로 YoungCha로 확정하였습니다.

- Youngcha Service
